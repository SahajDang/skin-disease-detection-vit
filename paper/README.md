## Abstract

This work proposes an advanced deep learning framework for automated skin disease classification using **Vision Transformers (ViT)** on the **ISIC 2018 dataset** consisting of **10,015 dermoscopic images across 7 disease classes**.

The proposed framework integrates:

* Vision Transformer (ViT-B16)
* Micro Fine-Tuning
* Contrastive Feature Learning
* Latent Space Hard Negative Mining
* Norm-Softmax Classification
* Focal Loss Optimization
* Test-Time Augmentation (TTA)

The final model achieved an accuracy of **84.16%**, outperforming CNN baselines by **17.16%**, while maintaining strong explainability and robustness.
